# Проект: **Mesto*

**Интро**

Проект - интерактивный сервис.
Где можно выкладывать фотокарточки с путешествий/отдыха, чтобы поделиться этим с миром, ставить лайки.
**Обзор использованных инструментов при создании проекта**
* Grid layout
* Flexbox
* media queries
* form layout
* popup box
